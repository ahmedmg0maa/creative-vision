# Creative Vision — Landing Website (Static)

## تشغيل محليًا
- افتح المجلد على VS Code
- افتح `index.html` مباشرة أو استخدم Live Server Extension

## نشر على GitHub Pages
1) ارفع الملفات على ريبو GitHub
2) Settings → Pages
3) Build and deployment: Deploy from a branch
4) اختر branch: main و folder: /root
5) هتطلع لك لينك الموقع

## تعديل سريع
- النصوص: `index.html`
- الألوان/التنسيق: `assets/css/styles.css`
- التفاعلات: `assets/js/main.js`
- اللوجو: `assets/img/logo.jpeg`
